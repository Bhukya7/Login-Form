# Login Form

## Overview
This project is a simple and responsive login form created using HTML, CSS, and Font Awesome icons. The form allows users to enter their credentials and provides options for social media login. It is designed to be visually appealing and user-friendly.

## Features
- User-friendly interface for logging in.
- Input fields for username and password.
- Social media login options (Facebook, Twitter, Google).
- Responsive design that adapts to various screen sizes.
- Eye-catching styling using CSS and Font Awesome icons.

## Technologies Used
- **HTML**: For structuring the login form.
- **CSS**: For styling the form and enhancing the user experience.
- **Font Awesome**: For adding icons to input fields and buttons.

## How It Works
1. The user is presented with a login form that includes fields for username and password.
2. Users can click on social media icons to log in using their social accounts.
3. A "Forgot password?" link is provided for users who need assistance with their credentials.
